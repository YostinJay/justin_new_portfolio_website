== Description ==

Schism is a free blog theme with clean type and a unique layout that makes it perfect for a personal blog.

Lightweight, completely responsive and SEO friendly, Schism is perfect for a personal blog with focus on your content displayed with a unique style.

Posts in the main loop display the full content, the footer have four widget areas and a sidebar can also be displayed in the single post.

The theme fully support the new block editor, the custom logo upload and have a custom color picker for the accent color.

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in Schism in the search form and press the 'Enter' key on your keyboard.
3. Click on the 'Activate' button to use your new theme right away.
4. Navigate to Appearance > Customize in your admin panel and customize to taste.
